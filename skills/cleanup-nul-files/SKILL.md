# Cleanup NUL Files

Удаляет артефактные файлы `nul`, которые могут появляться в проектах на Windows.

## Шаг 1: Найти все файлы `nul`

Используй `Glob` с двумя паттернами для поиска:

- `**/nul`
- `**/NUL`

Если ни один файл не найден — сообщи пользователю что файлов `nul` нет и заверши.

## Шаг 2: Удалить найденные файлы

Для каждого найденного файла выполни:

```bash
rm -rf "./path/to/nul"
```

Замени `./path/to/nul` на фактический путь к файлу из результатов Glob.

## Шаг 3: Подтвердить удаление

Повторно выполни `Glob` с теми же паттернами (`**/nul` и `**/NUL`).

Если результат пустой — сообщи пользователю что все файлы `nul` успешно удалены.
Если файлы остались — сообщи какие именно не удалось удалить.
